# חיבור Looker Studio לנתוני גרסאות לעם

**גרסאות לעם** (over.org.il) הוא מיזם לשקיפות שלטונית שעוקב אחרי מאגרי מידע
ממשלתיים בישראל ושומר את היסטוריית הגרסאות שלהם. המחבר (Community Connector)
מזרים כל טבלה מהמאגר — או כל שאילתת SQL — ישירות לתוך Looker Studio של גוגל,
כך שכל אחד יכול לבנות דשבורדים חיים על נתונים ציבוריים: מאגרי data.gov.il,
מסד הנתונים של הכנסת, החלטות ממשלה ועוד.

- **חינם**, ללא הרשמה — דרוש רק חשבון Google
- **קריאה בלבד** — שאילתות SELECT על נתונים ציבוריים
- **שמות עמודות בעברית** מוצגים כמו שהם
- עד **50,000 שורות** לשאילתה

## איך מתחברים

**[➕ הוספת המחבר ל-Looker Studio](https://lookerstudio.google.com/datasources/create?connectorId=AKfycbysYqW5DXT1fSzmMeOEN6lQfOCz-CF8gTA3rDcyqi7tk59buf9MEmMDpvGnFZ0AVqyVAA)**

1. **פותחים את קישור ההוספה** ומתחברים לחשבון Google אם מתבקשים.
2. **מאשרים את המחבר (פעם אחת).** בהפעלה הראשונה גוגל מציגה מסך אישור. ההרשאה
   היחידה שהמחבר מבקש היא פנייה לכתובת חיצונית אחת — over.org.il; הוא אינו ניגש
   לקבצים, למייל או לכל מידע אחר בחשבון.
3. **בוחרים מה להציג.** ברשימה הנפתחת "טבלה" בוחרים מאגר מהקטלוג (יש חיפוש), או
   כותבים שאילתת SELECT חופשית בתיבת ה-SQL — כולל JOIN בין מאגרים. דוגמה:

   ```sql
   SELECT "סטטוס חברה", COUNT(*) AS "מספר חברות"
   FROM append_ica_companies_cc6286ac
   GROUP BY 1 ORDER BY 2 DESC
   ```

   טיפ: מומלץ לנסות את השאילתה קודם בקונסולת ה-SQL שבאתר —
   [over.org.il/data](https://www.over.org.il/data) — שם יש רשימת טבלאות מלאה,
   השלמה אוטומטית ודוגמאות.
4. **לוחצים CONNECT ובונים.** מסך השדות מציג כל עמודה בשמה העברי ובטיפוס שזוהה;
   משם Create Report — וזה Looker Studio רגיל: גרפים, פילטרים ושיתוף. הצופים
   בדוח רואים את הנתונים בלי שום פרטי התחברות.

## מגבלות

| מגבלה | ערך | הערה |
|---|---|---|
| שורות לשאילתה | 10,000 (עד 50,000) | עדיף לסכם ב-SQL מאשר למשוך שורות גולמיות |
| זמן ריצה | 30 שניות | מצמצמים עמודות או מוסיפים WHERE |
| סוג שאילתות | SELECT / WITH בלבד | כתיבה חסומה ברמת מסד הנתונים |
| רענון נתונים | 12 שעות (ברירת מחדל) | מומלץ להשאיר — הנתונים מתעדכנים לפי קצב המקור |
