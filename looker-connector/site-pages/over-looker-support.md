# תמיכה — מחבר Looker Studio של גרסאות לעם

נתקלתם בבעיה בחיבור או בדוח? כאן תמצאו פתרונות לתקלות הנפוצות ודרכי פנייה.

## תקלות נפוצות

**שיניתי את ה-SQL והגרפים הפסיקו לעבוד** — פתחו את מקור הנתונים ב-Looker Studio
‏(Resource ← Manage added data sources ← Edit) ולחצו **REFRESH FIELDS**. רשימת
השדות תתעדכן לפי השאילתה החדשה; שדות שנשארו באותו שם ימשיכו לעבוד.

**שגיאת SQL** — ההודעה כוללת את פירוט השגיאה מהשרת. הסיבות השכיחות: שם טבלה או
עמודה שגויים, וגרשיים חסרים סביב שמות עמודות בעברית (יש לכתוב `"שם חברה"` ולא
`שם חברה`). מומלץ לבדוק את השאילתה קודם בקונסולה שבאתר:
[over.org.il/data](https://www.over.org.il/data).

**השאילתה נכשלת על זמן ריצה (timeout)** — מגבלת הריצה היא 30 שניות. צמצמו את
מספר העמודות, הוסיפו תנאי `WHERE`, או סכמו את הנתונים ב-`GROUP BY` במקום למשוך
שורות גולמיות.

**שגיאת מכסה (429)** — כלל הדשבורדים חולקים מכסת תעבורה יומית הוגנת. נסו שוב
מאוחר יותר; אם אתם צריכים היקף גדול יותר לפרויקט לגיטימי — כתבו לנו ונרחיב.

**מסך "Google hasn't verified this app"** — הודעה סטנדרטית למחברים קהילתיים.
ההרשאה היחידה שהמחבר מבקש היא פנייה ל-over.org.il; לחצו Advanced ואז
Go to OVER Looker Connector.
